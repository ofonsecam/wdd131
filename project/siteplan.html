Aura Planner: Site Plan

Oscar Fonseca - WDD 131

Site Name

Aura Planner

I chose this name because it evokes a sense of calm and clarity, which is essential for a task management tool. The goal is for the user to feel in control of their personal "aura" or environment.

Site Purpose

Aura Planner is a personal task manager designed as a Frontend MVP. Its purpose is to provide a clean, fast interface for users to capture and manage daily tasks without distractions, using local data persistence (localStorage). The site also includes a Contact page so users can reach the developer with suggestions or questions.

Scenarios

  • "I just remembered I need to buy materials for my BYU course; how can I quickly add this task and ensure it won't be deleted when I close the browser?"
  • "I have finished my first task of the day; where can I view my current list to remove the completed item and stay organized?"
  • "How can I reach the developer if I have a suggestion for Aura Planner?"

Color Schema

The palette is inspired by productivity and serenity:

Primary: #2c3e50 (Midnight Blue) - Used for headings and navigation.
Secondary: #ecf0f1 (Clouds) - Used for backgrounds and reading areas.
Accent: #27ae60 (Nephritis Green) - Used for action buttons and confirmations.

Typography

Montserrat: For headings (H1, H2, H3) to provide a modern and professional look.

Open Sans: For body text and task lists, prioritizing readability on mobile screens.

Wireframes

Multi-Page Site Structure

  • Home: Landing page introducing Aura Planner, with a hero section, project introduction, and navigation links.
  • My Planner (App): The main app interface supporting adding, viewing, and deleting tasks.
  • Contact: Simple contact form for reaching the developer.

The layout is responsive, with a single-column stack on mobile and navigation header at the top on all pages. On desktop, content is centered in a max-width container. Navigation appears on every page, allowing users to move between Home, My Planner, and Contact pages.
